## Install Package
```
$ pip install --upgrade pip
$ pip install cython
$ pip install "numpy<17"
$ pip install imutils
$ pip install flask
$ pip install opencv-python
$ pip install opencv-contrib-python
```

## Run Application
```
$ python wandlab.cv.ide.py
```
